# kritik-saran

kritikan dan saran : 
- yang sabar menghadapi rekan kamar
- pembelajaran lebih terstruktur
- lebih kontrol perkembangan per murid
- lebih berwibawa sebagai pengajar
- lebih bersahabat => ustadz fuad
- bagi waktu khusus untuk murid
- lebih tegas dalam kasih tugas / sering mengingatkan
- beri tugas terstruktur -> dari mudah ke normal ke susah ke sangat susah ke veteran
- lebih tepat waktu, dan usahakan tidak skip kelas
- jelaskan tujuan pembelajaran
- membuat menarik pelajaran
- buat pelajaran interaktif
- perbaiki kesan pertama
- jangan membiarkan tugas tidak selesai

- sesi belajar:
	- tujuan pembelajaran
	- materi
	- diskusi tugas

- kurikulum 13 pertemuan
